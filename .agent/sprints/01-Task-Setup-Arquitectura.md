# Sprint 1: Setup e Infraestructura Base (Next.js)
**Objetivo:** Inicializar el entorno moderno y conectar las herramientas core reciclando las buenas prácticas estructurales de `charatools-eccomerce`.

**Dependencias previas:** Ninguna (Fase Inicial).
**Reglas Aplicables:** Utilizar estándares del repo `antigravity-awesome-skills`. 

**Acciones:**
1. Crear la base del proyecto con Next.js (App Router preferiblemente) y Tailwind CSS.
2. Configurar la carpeta de variables de entorno para las credenciales de Supabase.
3. Crear un cliente `supabase` reutilizable en toda la app.
4. Instalar y configurar un gestor de estado moderno (preferiblemente Zustand) revisando si se utilizó en Charatools para mantener el mismo paradigma pero más optimizado.

**Criterios de Éxito:** Un proyecto inicializado corriendo sin errores, con una estructura de carpetas limpia (separando `/components`, `/lib`, `/store`) lista para escalar.
